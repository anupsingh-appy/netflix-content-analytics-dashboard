# Netflix Content Analytics Dashboard Using Tableau

## Project Overview

This project presents an interactive Netflix Content Analytics Dashboard built using Tableau Public. The dashboard analyzes Netflix movies and TV shows to uncover trends, content distribution, genre insights, ratings analysis, and country-wise production patterns.

The project focuses on data visualization, dashboard storytelling, and interactive business insights for Data Analyst portfolio development.

---

## Objectives

- Analyze Netflix content distribution
- Compare Movies vs TV Shows
- Identify top contributing countries
- Analyze ratings and genres
- Understand Netflix growth trends over time
- Build an interactive dashboard for business insights
